#  StockVision AI

## Machine Learning-Powered Stock Market Prediction & Technical Analysis Platform

StockVision AI is an intelligent stock market analytics platform that combines **Machine Learning, Technical Analysis, and Interactive Data Visualization** to help users analyze market trends and generate data-driven trading insights.

Built using **Python, Streamlit, Scikit-Learn, Plotly, Pandas, and NumPy**, the application predicts future stock closing prices, evaluates market risk, and provides actionable trading signals through a modern interactive dashboard.

##  Key Features

###  Stock Price Prediction

* Predicts the next closing stock price using a trained Machine Learning model.
* Utilizes historical stock market data and engineered technical indicators.
* Provides real-time prediction capabilities through an interactive dashboard.

###  Technical Analysis

The platform calculates multiple technical indicators including:

* 7-Day Moving Average (MA7)
* 30-Day Moving Average (MA30)
* Relative Strength Index (RSI)
* Daily Return Analysis
* Bollinger Band Position
* Market Volatility Metrics

###  Intelligent Trading Signals

Generates automated trading recommendations:

* 🟢 Strong Buy
* 🟢 Buy
* 🟡 Hold / Watch
* 🔴 Sell
* 🔴 Strong Sell

###  Risk Assessment

Evaluates investment risk using:

* Confidence Intervals
* Volatility Analysis
* Market Trend Evaluation

Risk Levels:

* Low Risk
* Medium Risk
* High Risk

###  Interactive Dashboard

* Candlestick Charts
* Price Trend Analysis
* Signal Strength Gauge
* Market Indicator Dashboard
* Real-Time Metrics

##  Project Workflow

### 1️ Data Collection

Historical stock market data is collected and prepared for analysis.

### 2️ Data Preprocessing

* Missing value handling
* Feature preparation
* Data transformation

### 3️ Feature Engineering

Technical indicators are generated to improve model performance.

### 4️ Model Training

Machine Learning models are trained using historical stock market patterns.

### 5️ Prediction & Analysis

The trained model generates future price predictions and trading insights.

### 6️ Visualization

Interactive charts and dashboards present market trends and predictions.
